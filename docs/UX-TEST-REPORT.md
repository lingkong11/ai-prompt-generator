## 四、交互设计验证

### 4.1 游客体验流程

```
访问 http://localhost:3002/
    ↓
显示完整主页 ✅
    ├── Header: [登录] [注册] 按钮 ✅
    ├── TAB: 提示词生成 / 模板库 / OpenClaw专用 / 收藏 ✅
    └── 可填写生成表单 ✅

点击「模板库」TAB
    ↓
正常展示模板卡片 ✅

点击「OpenClaw」TAB
    ↓
正常展示九阶段卡片 ✅

点击「收藏」TAB
    ↓
弹出登录/注册弹窗 ✅
```

### 4.2 认证弹窗验证

| 场景 | 操作 | 预期 | 结果 |
|------|------|------|------|
| 游客点击「生成」按钮 | 填写目标后点击生成 | 弹出认证弹窗 | ✅ PASS |
| 弹窗切换模式 | 点击「注册」TAB | 切换为注册表单 | ✅ PASS |
| 注册新用户 | 填写表单提交 | 注册成功，自动登录，弹窗关闭 | ✅ PASS |
| 登录后继续 | 登录成功 | 自动执行原操作（生成提示词） | ✅ PASS |

### 4.3 已登录用户体验

| 场景 | 操作 | 预期 | 结果 |
|------|------|------|------|
| 刷新页面 | F5 刷新 | 保持登录态，用户名显示 | ✅ PASS |
| 点击「生成」 | 直接操作 | 无弹窗，直接生成 | ✅ PASS |
| 点击「收藏」TAB | 切换TAB | 无弹窗，显示收藏列表 | ✅ PASS |
| 退出登录 | 点击用户下拉→退出 | 回到游客模式 | ✅ PASS |
