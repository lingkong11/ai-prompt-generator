# AI Prompt Generator - Sprint 2 定价模块
## 需求故事文档 (PRD) + UI设计规范

**版本**: v1.0  
**日期**: 2026-04-08  
**迭代**: Sprint 2  
**状态**: 需求分析阶段

---

## 一、需求背景与目标

### 1.1 背景
AI Prompt Generator 已完成核心功能（提示词生成、管理、收藏、国际化、主题切换），进入商业化变现阶段。

### 1.2 目标
- 建立清晰的付费订阅体系
- 引导免费用户向付费转化
- 支持多种支付方式（后续扩展）
- 提供灵活的套餐选择

---

## 二、用户故事 (User Stories)

### US-001 作为游客，我想查看定价页面
**验收标准**:
- 无需登录即可访问 /pricing 页面
- 清晰展示各套餐对比
- 突出推荐套餐（Pro）

### US-002 作为免费用户，我想升级套餐
**验收标准**:
- 点击升级按钮跳转登录（如未登录）
- 登录后直接跳转支付流程
- 支付成功后套餐立即生效

### US-003 作为付费用户，我想查看当前套餐
**验收标准**:
- 个人资料页显示当前套餐
- Header 显示套餐标识徽章
- 显示套餐到期时间

### US-004 作为付费用户，我想取消订阅
**验收标准**:
- 可在个人资料页管理订阅
- 取消后仍可使用至到期日
- 支持降级到免费版

### US-005 作为系统，我需要限制免费用户配额
**验收标准**:
- 每日生成次数限制（20次）
- 收藏数量限制（50条）
- 超出配额时引导升级

## 三、功能需求

### 3.1 套餐体系

| 套餐 | 价格 | 生成次数 | 收藏上限 | 模板库 | 导出 | 优先支持 |
|------|------|----------|----------|--------|------|----------|
| **Free** | ¥0/月 | 20次/日 | 50条 | 基础 | ❌ | ❌ |
| **Pro** | ¥29/月 | 无限 | 无限 | 全部 | ✅ | ✅ |
| **Team** | ¥99/月 | 无限 | 无限 | 全部+自定义 | ✅ | ✅ |

### 3.2 定价页面功能
- 套餐卡片展示（3列对比）
- 月付/年付切换（年付8折）
- 功能对比表格
- FAQ 常见问题
- 企业定制 CTA

### 3.3 订阅管理
- 当前套餐展示
- 升级/降级操作
- 取消订阅
- 账单历史（后续迭代）

### 3.4 配额限制
- 生成次数计数器
- 收藏数量检查
- 配额耗尽提示弹窗

## 五、技术需求

### 5.1 后端接口

```
GET  /api/subscription/plans          # 获取套餐列表
GET  /api/subscription/current        # 获取当前用户订阅
POST /api/subscription/upgrade        # 升级套餐
POST /api/subscription/cancel         # 取消订阅
GET  /api/quota/status                # 获取配额使用情况
```

### 5.2 数据库模型

```sql
-- 套餐表
subscription_plan (
  id, name, code, price_monthly, price_yearly,
  generate_limit, favorite_limit, features_json, is_active
)

-- 用户订阅表
user_subscription (
  id, user_id, plan_id, status, started_at, expires_at,
  payment_provider, payment_id, canceled_at
)

-- 配额使用记录表
quota_usage (
  id, user_id, usage_date, generate_count, favorite_count
)
```

### 5.3 前端组件

```
views/
  PricingView.vue          # 定价页
components/pricing/
  PricingCard.vue          # 套餐卡片
  PricingToggle.vue        # 月付/年付切换
  FeatureTable.vue         # 功能对比表
  PricingFAQ.vue           # FAQ组件
  QuotaAlert.vue           # 配额提示弹窗
stores/
  subscription.ts          # 订阅状态管理
```

## 六、非功能需求

### 6.1 性能
- 定价页面首屏加载 < 1s
- 套餐切换无刷新

### 6.2 安全
- 支付接口 HTTPS 强制
- 用户套餐状态服务端校验
- 防止配额绕过

### 6.3 可访问性
- 定价卡片支持键盘导航
- 价格信息屏幕阅读器友好
