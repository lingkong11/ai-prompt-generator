# AI Prompt Generator - 项目需求文档

## 一、项目概述

**项目名称**: AI Prompt Generator  
**项目类型**: Web应用（前后端分离）  
**核心定位**: 专业的AI提示词生成与管理平台，支持OpenClaw专用提示词  
**目标用户**: AI开发者、内容创作者、需要优化提示词的用户

### 1.1 核心功能

1. **提示词生成** - 输入目标，智能生成优化提示词
2. **模板库** - 预设多种类型提示词模板
3. **OpenClaw专用** - 提供AI Agent工作流专用提示词
4. **提示词测试** - 一键发送到AI服务测试效果
5. **收藏管理** - 收藏和分类管理提示词

### 1.2 差异化亮点（对比参考网站）

| 功能 | 参考网站 | 本项目亮点 |
|------|----------|------------|
| 提示词类型 | 通用 | ✅ 增加OpenClaw专用提示词 |
| 测试功能 | 无 | ✅ 集成AI服务测试 |
| 模板系统 | 基础 | ✅ 分类模板+自定义模板 |
| Agent能力 | 无 | ✅ 生成可直接执行的Agent任务 |

## 二、技术设计

### 2.1 技术栈

- **后端**: Spring Boot 3.x + Java 17
- **前端**: Vue 3 + TypeScript + Element Plus
- **数据库**: PostgreSQL（可选，本地模式用H2）
- **AI集成**: OpenAI API / DeepSeek API

### 2.2 项目结构

```
ai-prompt-generator/
├── server/                 # 后端服务
│   ├── src/main/java/com/ai/prompt/
```

## 三、功能模块

### 3.1 提示词生成模块

**输入**:
- 目标描述（如"写一篇关于AI的文章"）
- 提示词类型（写作/编码/分析/Agent）
- 语气风格（正式/随意/技术）
- 输出语言（中文/英文）

**输出**:
- 优化后的提示词
- 使用建议

### 3.2 OpenClaw专用模块

**提示词类型**:
- 需求分析提示词
- 技术设计提示词
- 代码开发提示词
- 测试执行提示词
- 文档输出提示词
- 项目交付提示词

### 3.3 模板库模块

**预置模板**:
- 写作助手模板
- 代码审查模板
- 会议纪要模板
- 翻译助手模板
- OpenClaw九阶段模板

### 3.4 收藏管理模块

- 收藏提示词
- 分类管理
- 搜索过滤
